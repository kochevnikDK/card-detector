# 🃏 Детектор и распознаватель игральных карт

Программа для обнаружения и распознавания игральных карт на изображениях с использованием современных методов компьютерного зрения.

## ✨ Возможности

- **3 метода детекции**: YOLO (быстрый), OCR (точный), Гибридный (оптимальный)
- **Графический интерфейс** для калибровки параметров в реальном времени
- **Асинхронная обработка** - интерфейс не зависает
- **Визуализация** всех этапов распознавания
- **Кэширование результатов** для ускорения повторной обработки
- **Поддержка шаблонов** для улучшения OCR
- **Сохранение/загрузка** настроек

## 🚀 Быстрый старт

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/yourusername/card_detector.git
cd card_detector

# Создание виртуального окружения (рекомендуется)
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/Mac

# Установка зависимостей
pip install -r requirements.txt

## Запуск

# Запуск графического интерфейса
python calibrate.py

# Или через main.py
python main.py --gui

# Обработка одного изображения
python main.py --input input/cards.jpg --output output/result.jpg
